## About the Project

**Odin Landing Page** is a project created to practice fundamental web development skills, focusing on layout and styling using HTML and CSS.

The purpose of this repository is to build a structured landing page using core technologies, focusing on:

* Proper HTML structure
* CSS Flexbox layout
* Visual hierarchy and spacing
* Working with images and colors
* Organizing styles with external CSS

This project does not use JavaScript or CSS frameworks. The goal is to strengthen foundational knowledge of HTML and CSS before moving on to more advanced tools.

## Page Sections

The landing page contains the following sections:

* **Header / Hero**: Navigation bar and a hero section with a headline, subtext, and a call-to-action button.

* **Info / Cards Section**: A row of cards showcasing images with short descriptions below each one.

* **Quote / Highlight Section**: An inspirational or featured quote section with a distinct background.

* **Call to Action Section**: A section encouraging user interaction, with a button linking to further content.

* **Footer**: Simple footer with copyright or contact information.

   git clone https://github.com/willaevangelista/odin-landing-page.git
2. Open the project folder.
3. Open `index.html` in your browser.

No build step or dependencies are required.
